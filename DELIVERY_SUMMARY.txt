### üöÄ Steps for the CLIENT (Deployment):
1. **Upload**: Upload the contents of the ZIP to their server (cPanel or VPS).
2. **Environment (.env)**: 
   - Rename `.env.example` to `.env`.
   - Update database credentials (`DB_DATABASE`, `DB_USERNAME`, `DB_PASSWORD`).
   - Set `APP_URL` to their domain.
3. **Run Commands**:
   - `composer install --no-dev`
   - `php artisan key:generate`
   - `php artisan migrate` (Run this to update the table structure. Do NOT run with `--seed` if you already have data).
4. **Storage Link**: Run `php artisan storage:link` (or use the included `fix-storage-symlink.bat`) to ensure photos display correctly.
